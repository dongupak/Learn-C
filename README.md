# "따라하며 배우는 C 언어" 저장소 소개

안녕하세요 독자여러분 이곳은 생능출판사의 *"따라하며 배우는 C 언어" 관련 저장소입니다.
이 책의 소스 코드와 정오표등 다양한 자료와 정보를 업로드할 예정입니다.

이 책은 C 언어을 처음 배우는 입문자와 다른 프로그래밍 언어를 익힌 상태에서 새롭게 C 언어에 도전하는 개발자를 위한 책입니다.
이 책은 다음과 같은 특징이 있습니다.
#### 첫째, C 언어의 문법에 대한 설명과 함께 이러한 문법의 개념에 대해서 상세히 설명합니다
#### 둘째, C 언어를 입문하는 독자들을 위한 친절하고 상세한 설명과 많은 예시 그림이 있습니다.
#### 셋째, 각 장은 핵심 주제에 맞는 짧은 절로 나뉘어져 있어 간결하게 핵심을 파악할 수 있습니다
#### 넷째, 독자들이 이해한 내용을 스스로 연습하고 확인해 볼 수 있는 많은 LAB과 도전문제들이 있습니다.
#### 다섯째, 핵심 정리를 통해서 각 장의 내용을 정리하고 익힐 수 있도록 하였습니다.
#### 여섯째, 주관식 문제와 심화문제, 프로그래밍 문제를 통해 스스로 문제 해결을 할 수 있는 기회를 제공합니다.
